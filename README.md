# Marina Litvinenko Site

Статический сайт-визитка для психолога Марины Литвиненко.

## Структура

- `docs/` — финальная версия сайта для GitHub Pages
- `marina-litvinenko-site-mockup/` — исходный локальный макет
- `marina-litvinenko-tilda-outline.md` — текстовый каркас страницы

## Публикация на GitHub Pages

1. Создать новый репозиторий на GitHub.
2. Инициализировать git локально, если нужно.
3. Добавить файлы и сделать первый коммит.
4. Подключить удалённый репозиторий.
5. Запушить ветку `main`.
6. В настройках GitHub Pages выбрать публикацию из ветки `main` и папки `docs/`.

## Что ещё нужно перед реальной публикацией

- добавить реальные контакты в блок записи
- подключить рабочую форму или заменить её на Telegram / WhatsApp / email
- при необходимости подключить собственный домен
